Skyrim Setup

Buy Skyrim

Buy The Elder Scrolls V: Skyrim Anniversary Edition form Steam or Keyseller of your choice.

Install Skyrim

For a clean install remove following directories if available:

  • C:\Program Files (x86)\Steam\steamapps\common\Skyrim Special Edition
  • C:\Users\user\Documents\My Games\Skyrim Special Edition
  • C:\Users\user\Documents\My Games\Skyrim.INI

Before installation through Steam go to Skyrim's properties and set your language. I recommend English, because of some modded UI elements for immersion purposes, but you can pick whatever you want. The language setting also changes the audio files for the game.

Preferably turn off Steam Cloud so your savegames are stored locally.

Be sure to enable the Anniversary Upgrade in the DLC's.

After that install Skyrim.

Afterwards you can verify game files to be safe.

Launch Skyrim

Launch Skyrim once and let it download all the Creation Club Creations.

After the download has finished go to Creations and press O or select the Options button. Double check your downloads by pressing "Download all owned Creation Club Creations".

Close the game and navigate to your game files located at C:\Program Files (x86)\Steam\steamapps\common\Skyrim Special Edition\Data.

Check if you have 148 files in total starting with cc. These are the Creation Club Creations. If not try to verify game files or redownload the game completely.

Vortex Setup

Install Vortex

Install Vortex Mod Manager

Inside Vortex go to Settings>Interface and enable Profile Management.

Also set Downlaod Threads to 10 for more paralel downloads inside Settings>Download

Manage Skyrim

In Vortex click on Games and search for Skyrim Special Edition. Afterwards click on Manage to enable Vortex to install mods for this game.

Go to Profiles and click on "Add Skyrim Special Edition Profile" and call it as you like. Also check settings to allow this new Profile to have its own save games and game settings.

Save and and enable this profile.

Download Mods

Download the Mod Collection by following this link or pressing Add to Vortex.

Be sure to select the correct Profile in Vortex before downloading the mods.

Installing all mods depends on PC specs and download speed.

You can see the progress by clicking on Collections in Vortex and clicking View on Skyrim Together+ Updated.

After installation go to notifications and fix loose files.

Vortex Tools

Go to your Dashboard in Vortex and navigate to Tools. Create a new Tool and call it Multiplayer.

The Target should be set to

  • C:\program files (x86)\steam\steamapps\common\Skyrim Special Edition\Data\SkyrimTogetherReborn\SkyrimTogether.exe

depending on you installation directory. Press save and Run Skyrim from the new Shortcut in Tools.

Note: It can happen that you need to launch SkyrimTogether.exe manually once and select the SkyrimSE.exe.
